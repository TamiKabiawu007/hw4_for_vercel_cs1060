Welcome to HW4 For Vercel

This application exposes an endpoint for conversion.

Usage

To test the conversion endpoint, run the following curl command:

curl -H "Content-Type: application/json" \
  -d '{"input": "five", "inputType": "text", "outputType": "decimal"}' \
  "https://hw4-for-vercel-cs1060.vercel.app/convert"

If your request is successful, you should receive a JSON response.
