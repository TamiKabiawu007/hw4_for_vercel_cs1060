County Data API

This API provides access to county health data. You can query various health measures by ZIP code.

Available Measures

  • Violent crime rate
  • Unemployment
  • Children in poverty
  • Diabetic screening
  • Mammography screening
  • Preventable hospital stays
  • Uninsured
  • Sexually transmitted infections
  • Physical inactivity
  • Adult obesity
  • Premature Death
  • Daily fine particulate matter

API Usage

To get county data, make a POST request to the /county_data endpoint:

curl -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "zip": "12345",
    "measure_name": "Adult obesity",
    "limit": 10
  }' \
  "https://hw4-for-vercel-cs1060-jd4fhilgx-tamikabiawus-projects.vercel.app/county_data"

Request Parameters

  • zip: A 5-digit ZIP code (required)
  • measure_name: One of the measures listed above (required)
  • limit: Maximum number of results to return (optional, defaults to 10)

The API will return a JSON array of county data matching your query:
